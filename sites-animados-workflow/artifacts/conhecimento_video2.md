## Resumo

Demonstração prática de como criar landing pages cinematográficas **estilo Apple** em menos de 10 minutos usando IA. O autor recria a página do teclado **Logitech Master 3S** com um efeito de **desconstrução 3D na Hero Section** e scroll-triggered video, usando um pipeline 100% com ferramentas de IA.

---

## 🛠️ Stack Completo (em ordem de uso)

### 1. Nano Banana 2 — Geração de Imagens IA
- Gera **imagens de produto de alta qualidade** a partir de prompts
- Usada para criar as imagens do teclado Logitech com fundo premium
- Controla ângulo, iluminação, estilo e acabamento da imagem
- **Timestamp:** 02:29

### 2. Kling 3.0 — Animação de Vídeo com IA
- Recebe a imagem gerada e cria uma **animação de transição/movimento**
- Gera vídeo a partir de uma imagem estática (image-to-video)
- Cria o efeito de "desconstrução 3D" que será sincronizado com o scroll
- **Timestamp:** 06:29

### 3. Antigravity — Programação do Site
- Principal ferramenta de **desenvolvimento do site com IA**
- Usa **Firecrawl** para fazer scraping do site original (Logitech) e usar como referência
- Gera a estrutura HTML/CSS/JS do site
- Implementa o **efeito de scroll** que sincroniza com o vídeo
- **Timestamp:** 07:03

### 4. FFMPEG — Otimização de Vídeo
- Ferramenta CLI gratuita para **compressão e otimização de vídeos**
- Resolve problema de **travamento (lag)** de vídeos pesados no browser
- Converte para formatos otimizados para web
- **Timestamp:** 11:15

---

## 🎬 Técnica Principal: Scroll-Triggered Video

```
COMO FUNCIONA O EFEITO:
1. Usuário está no topo da página → vídeo no frame 0
2. Usuário rola para baixo → vídeo avança proporcionalmente
3. Usuário chegou ao fim da seção → vídeo no último frame

RESULTADO: A "desconstrução 3D" do produto acontece conforme o scroll
```

Este é o efeito mais impactante do site estilo Apple (como em apple.com/iphone).
